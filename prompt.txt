Quality Assurance – Mandatory 10-Mark Answer Framework (Expanded):
 
For EACH question, the answer MUST follow this structure IN ORDER and IN FULL DETAIL:
 
 
1.  
Introduction (Minimum 3–5 lines)
 
• Clearly define key terms.
 
• State the purpose and importance of the concept.
 
• Briefly mention where the concept is used or applied.
 
 
2.  
In-depth Analysis / Core Explanation (Minimum 3–5 solid paragraphs)
 
• Explain all underlying principles step-by-step.
 
• Break complex ideas into sub-points or stages.
 
• Use proper academic tone and terminology.
 
• Include advantages, limitations, assumptions, or conditions where applicable.
 
 
3.  
Equations / Mathematical Representation (Wherever relevant)
 
• Clearly write all relevant formulas.
 
• Define EVERY variable used.
 
• Explain what the equation represents conceptually.
 
• Do NOT skip explanation assuming prior knowledge.
 
 
4.  
Diagram / Visual Representation (Mandatory wherever applicable)

• Clearly label components.
 
• Explain the diagram in words.
 
• If a diagram is not applicable, explicitly state why.
 
 
5.  
Example / Application (Mandatory)
 
• Provide a realistic, exam-appropriate example.
 
• Can be numerical, conceptual, or real-world.
 
• Clearly connect the example to the theory explained above.
 
 
6.  
Conclusion (Minimum 3–4 lines)
 
• Summarize the core idea.
 
• Reinforce why the concept is important.
 
• End with a strong academic takeaway suitable for exam answers.
 
 
7.  
Mini-Audit (Internal – Do NOT display reasoning)
 
• Internally verify:
 
"Is this answer detailed enough for a full 10 marks?"
